<h1 align="center">
Btxpack
</h1>
<h3 align="center">
Bare Texture Packer
</h3>

## Install
```
go install github.com/intervinn/btxpack/cmd/btxpack@latest
```

## Usage
Consumes folder of images, emits combined image and the metadata file.
```bash
btxpack [assets] [atlas.png] [atlas.json]
```
